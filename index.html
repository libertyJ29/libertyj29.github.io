Construction Quote App

Responsive web app for the Management of client's construction quotes

Magento 2 Backend Module

Module for analsis of an e-commerce company's sales

Denstist Appointment Scheduling

Manage Daily patient appointments

Appointment Scheduling System for a ‘Paper Based’ Dentist that manages daily patient appointments and all associated patient and dentist records.

The system was developed with laravel. jQuery was used for the main appointment scheduler buttons and for some small components.

AngularJS was used for updating the screen instantly such as to allow the user to add a new treatment type for a dentist and then select it straight away without refreshing/leaving the screen, or to verify that the date seleted in the main appointment scheduler is actually available(by querying the database) on the appointment booking screen.

Technologies: AngularJS, Laravel, jQuery, Javascript, MySQL.
